## ğŸš¨ If Authentication Doesn't Work

If signup/login fails on your custom domain:

### Common Issues:

**Issue**: "Invalid login credentials"  
**Cause**: Supabase URLs not updated  
**Fix**: Update Site URL and Redirect URLs in Supabase (see above)

**Issue**: "Redirect URL not whitelisted"  
**Cause**: Missing redirect URL in Supabase  
**Fix**: Add all 4 redirect URLs in Supabase

**Issue**: Session doesn't persist  
**Cause**: Cookie domain mismatch  
**Fix**: Clear browser cache and cookies, try again

**Issue**: CORS error  
**Cause**: Supabase CORS settings  
**Fix**: Wait 5-10 minutes after updating URLs
